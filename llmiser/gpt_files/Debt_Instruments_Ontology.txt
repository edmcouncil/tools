
Call event is defined as an event associated with the call schedule for a debt instrument. Call event is a kind of redemption event.
Call feature is defined as redemption provision defining the rights of the issuer to buy back a security at a call price after a call protection period. Call feature has synonyms call provision. Call feature is a kind of redemption provision. It comprises some call schedule. It has notification provision at least zero call notification provision.
Call notification provision is defined as provision of a call feature defining notification requirements. Call notification provision is a kind of notification provision. It has date period only call window.
Call premium is defined as the price over par paid by an issuer to redeem securities when exercising a call provision. Call premium is a kind of monetary price.
Call schedule is defined as a schedule of call prices and when they are in effect. Call schedule is a kind of schedule. It comprises some call event. It comprises some that monetary price or call premium.
Call window is defined as an explicit period of time prior to a call date during which an issuer or agent must give notice to holders. Call window is a kind of date period.
Debt offering is defined as an offering of one or more debt instruments for sale. Debt offering is a kind of securities offering. It applies to some debt instrument. It applies to some tradable debt instrument. It has offering amount at most one monetary amount. It is evidenced by some prospectus. It is underwritten by some security underwriter.
Fixed income security is defined as tradeable debt instrument that provides a return in the form of fixed periodic payments and typically the return of principal at maturity. Fixed income security is a kind of tradable debt instrument.
Fully-indexed interest rate is defined as a variable interest rate that is calculated by adding a margin to a specified index rate. Fully-indexed interest rate is a kind of expression, floating interest rate. It has argument exactly one base rate. It has argument some margin.
Interest-only strip is defined as a strip that represents the non-principal portion of the monthly payments on the underlying debt instrument, such as a bond. Interest-only strip has synonyms IO strip. Interest-only strip has examples An interest-only strip can be reintegrated into other synthetic or engineered products. For example, interest-only strips can be pooled to create or make up a portion of a larger collateralized mortgage obligation (CMO), asset-backed security (ABS) or collateralized debt obligation (CDO) structure.. Interest-only strip is a kind of strip.
Issued debt is defined as debt that is set out by the borrower in some form of financial security in which the lender is the holder or counterparty of that security. Issued debt is a kind of debt. It has debt amount only monetary amount. It is conferred by only debt instrument.
Make whole feature is defined as a call provision allowing the issuer to pay off remaining debt early. Make whole feature has synonyms make whole provision. Make whole feature is a kind of call feature.
Margin is defined as a variable that is added to a specified index rate to determine the fully indexed interest rate charged to a borrower on a credit balance. Margin has synonyms spread. Margin is a kind of variable.
Non-tradable debt instrument is defined as a debt instrument that may not be bought or sold. Non-tradable debt instrument has examples Low-risk instruments such as savings bonds are examples of nonnegotiable debt instruments.. Non-tradable debt instrument is a kind of debt instrument, mutual contractual agreement. It has redemption terms at least zero redemption provision. It has relative price at issue some relative price. It has relative price at redemption some relative price.
Notification provision is defined as provision of a redemption feature defining notification requirements. Notification provision is a kind of redemption provision. It has date period only date period.
Prescriptive event is defined as an event related to the imposition or enforcement of a rule, method, formula, etc.. Prescriptive event is a kind of occurrence kind.
Principal-only strip is defined as a strip that represents the principal portion of the monthly payments on the underlying debt instrument, such as a bond. Principal-only strip is a kind of strip.
Publicly issued debt is defined as an issued debt in the form of a tradable debt instrument (security). Publicly issued debt is a kind of issued debt. It is conferred by only tradable debt instrument.
Put event is defined as an event associated with the put schedule for a debt instrument, i.e., an event involving the 'put', or surrender of the instrument by the holder. Put event is a kind of redemption event.
Put feature is defined as redemption provision giving the holder the right, but not the obligation, to sell a specified amount of the debt instrument (i.e., redeem it), prior to maturity. Put feature has synonyms put provision. Put feature is a kind of debt terms, redemption provision. It comprises some put schedule. It has notification provision at least zero put notification provision. It is mandatory exactly one boolean.
Put notification provision is defined as a provision of a put feature defining notification requirements. Put notification provision is a kind of notification provision. It has date period only put window.
Put premium is defined as an amount over par that a debt instrument holder must pay to sell the security early. Put premium is a kind of monetary price.
Put schedule is defined as a schedule that defines the events associated with the put feature of a debt instrument, i.e, the dates on which the debt instrument may be sold at what price by the holder. Put schedule is a kind of schedule. It comprises some put event. It comprises some that monetary price or put premium.
Put window is defined as an explicit period of time prior to a put date during which holder or agent must give notice to an issuer. Put window is a kind of date period.
Redemption event is defined as a prescriptive event involving the return of an investor's principal in a fixed-income security, such as a preferred stock or bond, or the sale of units in a mutual fund. Redemption event is a kind of prescriptive event.
Relative price is defined as security price specified in comparison with either a stated or market value for a debt instrument at some point in time. Relative price is a kind of security price.
Strip is defined as a tradeable debt instrument created either through the process of removing coupons from a bond and then selling the separate parts as a zero coupon bond and an interest paying coupon bond or through taking the opposite position from some variant in the options market. Strip has synonyms Separate Trading of Registered Interest and Principal of Securities. Strip is a kind of fixed income security.
Tradable debt instrument is defined as a debt instrument that is also a security, i.e., that can be bought and sold by the holder. Tradable debt instrument is a kind of debt instrument, security. It has guarantor at least zero guarantor. It has yield some debt instrument yield. It is subordinated to at least zero tradable debt instrument.
Variable income security is defined as tradeable debt instrument that provide their owners with a rate of return that is dynamic and determined by market forces. Variable income security is a kind of tradable debt instrument.
