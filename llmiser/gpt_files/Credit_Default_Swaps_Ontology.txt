
Asset-backed credit default swap is defined as credit default swap whose underlying reference obligation is an asset-backed security rather than corporate credit. Asset-backed credit default swap is a kind of credit default swap.
Basket credit default swap is defined as credit default swap that references a bespoke, synthetic portfolio of underlying assets whose components have been agreed to for a specific OTC derivative by the parties to the transaction. Basket credit default swap is a kind of multi-name credit default swap.
Basket of credit default swaps is defined as basket of reference obligations are credit default swap instruments. Basket of credit default swaps is a kind of dated structured collection, weighted basket. It has constituent some credit default swap.
Cash settlement method is defined as strategy for calculating or otherwise establishing a reference final price for the contract. Cash settlement method is a kind of price determination method.
Contingent credit default swap is defined as credit default swap in which an additional triggering event is required. Contingent credit default swap is a kind of credit default swap.
Credit default swap is defined as bilateral contract in which one party (protection seller) agrees to provide payment to the other party (protection buyer) should a credit event occur against the underlying, which could be a specified debt (the reference obligation), a specific debt issuer (reference entity), a basket of reference entities and/or reference obligations, or a credit index (reference index). Credit default swap is a kind of credit derivative, swap. It has contract price some monetary price. It has contractual element some credit protection terms. It has occurrence at least zero settlement auction. It specifies some triggering event.
Credit default swap index is defined as index whose underlying reference obligations are credit default swap instruments. Credit default swap index is a kind of reference index. It is based on some basket of credit default swaps.
Credit event notice is defined as irrevocable written or verbal notice that states that a triggering event has occurred. Credit event notice is a kind of notice. It is about some triggering event. It is provided by some notifying party.
Credit protection terms is defined as legal terms that define triggering events and associated conditions related to settlement. Credit protection terms has synonyms contingent leg. Credit protection terms is a kind of derivative terms. It allows substitution at least zero boolean. It has buyer some deliverable obligation buyer. It has seller some deliverable obligation seller. It scheduled termination date at least zero explicit date. It specifies some deliverable asset. It specifies some triggering event.
Deliverable asset is defined as asset that must be delivered as a part of the process of settling a credit default swap. Deliverable asset is a kind of asset.
Deliverable obligation buyer is defined as contract party that is obliged to purchase a deliverable obligation (asset) if a triggering event occurs, depending on the event and the contract. Deliverable obligation buyer is a kind of buyer, contract party.
Deliverable obligation seller is defined as contract party that is obliged to sell a deliverable obligation (asset) if a triggering event occurs, depending on the event and the contract. Deliverable obligation seller is a kind of contract party, seller.
Escrow agent is defined as third party that holds an asset or funds before they are formally transferred from one party to another party, per the terms of a contract, within some specified time period and/or when a triggering event occurs. Escrow agent is a kind of contract third party, registered agent. It holds some deliverable asset.
Index credit default swap is defined as credit default swap that references a family of standardized credit derivative indices, where the underlying reference entities are a defined basket of credit from a particular geographic region (e.g. Asia, North America, Europe), and/or credit rating level (e.g. emerging markets, high yield, investment grade). Index credit default swap is a kind of multi-name credit default swap.
Index tranche credit default swap is defined as credit default swap that references a synthetic collateralized debt obligation (CDO) based on a credit index where each tranche references a different segment of the loss distribution of the underlying index. Index tranche credit default swap is a kind of multi-name credit default swap.
Loan credit default swap is defined as credit default swap whose underlying reference obligation is limited strictly to syndicated secured loans, rather than any type of corporate debt. Loan credit default swap is a kind of credit default swap.
Multi-name credit default swap is defined as credit default swap that references more than one obligation or borrower. Multi-name credit default swap has synonyms multiname credit default swap. Multi-name credit default swap is a kind of credit default swap.
Notifying party is defined as party responsible for issuing one or more formal notices indicating that an event that is relevant to a given contract has taken place. Notifying party is a kind of party in role. It contract party or contract third party.
Settlement auction is defined as independently administered synthetic auction process on a set of defined deliverable obligations that sets the reference final price that can be used to facilitate cash settlement of all covered transactions following a credit event. Settlement auction is a kind of settlement event.
Single name credit default swap is defined as credit default swap whose underlying risk is a single reference obligation, or a single reference entity, such as a corporation or a sovereign borrower. Single name credit default swap is a kind of credit default swap.
Triggering event is defined as event that relates to or triggers some aspect of a credit default swap. Triggering event is a kind of occurrence.
