
Basis swap is defined as swap in which payment streams are referenced to different bases. Basis swap is a kind of rates swap.
Correlation leg is defined as floating leg of a correlation swap that pays an amount based on the observed average correlation of the prices of a collection of underlying products. Correlation leg is a kind of performance-based variable leg. It is leg of some correlation swap.
Correlation swap is defined as over-the-counter statistical derivative that allows one to hedge risks associated with the observed average correlation of a collection of underlying products. Correlation swap is a kind of statistical swap.
Dispersion leg is defined as floating leg of a dispersion swap that pays an amount based on the realized dispersion of the price changes of the underlying product. Dispersion leg has synonyms variance leg. Dispersion leg is a kind of performance-based variable leg. It is leg of some dispersion swap.
Dispersion swap is defined as statistical derivative used to hedge on the magnitude of a price movement of an underlying asset. Dispersion swap has synonyms variance swap. Dispersion swap is a kind of statistical swap. It has leg exactly one dispersion leg.
Excess return swap is defined as return swap in which the change in the level of the index will be equal to the returns generated primarily by the changes in price of each of the contracts that comprise the index. Excess return swap is a kind of index return swap.
Fixed leg is defined as swap leg whose paying party is obligated to make payments from time to time over the term of a swap calculated by reference to a fixed annual rate or consisting of at least one payment of a fixed amount. Fixed leg is a kind of swap leg. It has payment schedule at least zero payment schedule.
Fixed payment leg is defined as swap leg that specifies contractual terms associated with a schedule of payments for any swap calculated by reference to a fixed annual rate. Fixed payment leg has synonyms fixed payment stream terms funding leg. Fixed payment leg is a kind of swap leg. It has payment schedule some payment schedule.
Floating leg is defined as swap leg whose paying party is obligated to make payments from time to time over the term of a swap calculated by reference to some variable rate, calculated based on the change in value of some underlying asset, or consisting of at least one payment of some floating amount. Floating leg is a kind of swap leg. It has payment schedule at least zero payment schedule.
Index return swap is defined as return swap in which payments are based on a fee paid to the seller of the swap and on a floating reference price based on changes in the level of an index from an initial level to a level observed on some valuation date(s). Index return swap is a kind of return swap.
Major swap participant is defined as financial service provider that maintains a substantial position in swaps for any of the major swap categories. Major swap participant is a kind of non-depository institution.
Performance-based variable leg is defined as floating leg of a swap that depends on some statistical measure of the performance of the underlier. Performance-based variable leg is a kind of floating leg. It is leg of some statistical swap.
Rate-based leg is defined as swap leg of a rate-based swap based on a floating interest, floating inflation or fixed interest rate. Rate-based leg is a kind of swap leg. It has underlier some rate-based observable. It is leg of some rates swap.
Rates swap is defined as swap in two counterparties each agree to pay the other cash flows on defined dates during an agreed period, based on a specified notional amount and a floating interest, floating inflation or fixed interest rate. Rates swap is a kind of swap. It has leg exactly 2 rate-based leg.
Realized variable leg is defined as performance-based leg wherein the payment is netted at maturity rather than periodically. Realized variable leg is a kind of performance-based variable leg.
Return leg is defined as leg of a swap in which the income generated by some underlier is paid. Return leg is a kind of floating leg. It has some underlier that is that tangible asset or reference index. It is leg of some return swap.
Return swap is defined as swap in which one leg, the return leg, is based on income generated from some underlier. Return swap is a kind of swap. It has funding leg some fixed payment leg. It has return leg some return leg.
Simple return leg is defined as return leg based strictly on the income generated by the underlying asset, excluding appreciation, decrease in value, capital gains, and fees on any increase in value. Simple return leg is a kind of return leg.
Statistical swap is defined as swap that depends on some statistical measure of the performance of the underlier. Statistical swap is a kind of swap. It has leg at least zero fixed leg. It has leg some performance-based variable leg.
Strike leg is defined as swap leg that specifies a fixed amount, 'the strike', quoted at the time of execution. Strike leg is a kind of fixed leg.
Swap is defined as derivative instrument whereby two counterparties agree to exchange periodic streams of cash flows with each other. Swap is a kind of derivative instrument. It exchanges exactly 2 swap leg. It has contract party exactly 2 swap party. It has contractual element some swap terms. It is identified by at least zero unique swap identifier.
Swap confirmation is defined as formal confirmation that codifies the terms and conditions specific to a lifecycle event with respect to the overall transaction between the parties. Swap confirmation is a kind of swap lifecycle event, transaction confirmation. It has contract party some swap party. It supersedes only swap confirmation.
Swap data repository is defined as financial service provider that provides a central facility for swap data reporting and recordkeeping. Swap data repository is a kind of financial service provider.
Swap dealer is defined as non-depository institution such as one that deals in swaps, makes a market in swaps, regularly enters into swaps with counterparties as an ordinary course of business for its own account, and engages in any activity causing the person to be commonly known in the trade as a dealer/market maker in swaps. Swap dealer is a kind of dealer, non-depository institution.
Swap leg is defined as terms defining and the commitment to fulfill cashflow requirements (e.g., interest payments, coupon payments, etc.) for one side of a swap. Swap leg has synonyms swap stream terms. Swap leg is a kind of cashflow terms, swap terms. It has buyer exactly one swap paying party. It has effective date exactly one date. It has notional amount exactly one monetary amount. It has seller exactly one swap receiving party. It has termination date at least zero date. It is leg of some swap.
Swap lifecycle event is defined as event that occurs during the lifecycle of a swap, where the overall set of events includes those specific to the issuer, front office, middle office, and/or back office processes. Swap lifecycle event is a kind of transaction event. It applies to some swap. It has party in role some swap party. It has swap identifier exactly one unique swap identifier. It is identified by exactly one swap lifecycle event identifier.
Swap lifecycle event identifier is defined as identifier that uniquely identifies a specific event in the lifecycle of a swap. Swap lifecycle event identifier is a kind of identifier. It identifies exactly one swap lifecycle event.
Swap party is defined as party to a swap and therefore a legal party to the contract that embodies that transaction. Swap party is a kind of contract party. It is played by some that is a party to some swap.
Swap paying party is defined as swap party responsible for making payments for a given leg of the transaction as defined in the contract. Swap paying party is a kind of paying party, swap party.
Swap receiving party is defined as swap party that receives payments for a given leg of the transaction as defined in the contract. Swap receiving party is a kind of receiving counterparty, swap party.
Swap stream event is defined as payment event (e.g., interest payment, coupon payment, etc.) against one leg of a swap stream. Swap stream event is a kind of payment event, swap lifecycle event.
Swap terms is defined as legal contractual terms of a swap transaction. Swap terms is a kind of derivative terms.
Total return leg is defined as return leg based on the income generated by the underlying asset as well as any appreciation, decrease in value, capital gains, and fees on any increase in value. Total return leg is a kind of return leg.
Total return swap is defined as return swap where the seller agrees to pay the other party the difference in value of some underlying asset multiplied by an agreed-upon notional value should the asset value increase between specified periods of time. Total return swap has examples For example the parties may enter into a two year agreement where every three months they compare the value of the Barclays Capital Aggregate Bond Index to its value three months previously. If the agreed upon notional was US $10,000,000 and the value increased 0.04%, or 4 basis points (bps), the seller would pay the buyer US $4,000. If, after another three months, the value decreased by 3bps, the buyer would pay the seller US $3,000. As part of the agreement, the buyer may also make an additional payment each period to the seller based on a floating rate index multiplied by the notional value.. Total return swap is a kind of credit derivative, return swap. It has return leg some total return leg.
Unique swap identifier is defined as identifier for a swap that is used in recordkeeping and swap data reporting. Unique swap identifier is a kind of financial instrument identifier. It identifies exactly one swap.
